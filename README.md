## Endpoints

* `/api/categories`
* `/api/articles`

## Dependencies

* .net core 3.0
* EntityFrameworkCore
* AutoMapper

## Use

* Clone
* Cd to root and `dotnet run`
* Test inside browser:
    *  `http://localhost:8081/api/categories`
    *  `http://localhost:8081/api/articles`
